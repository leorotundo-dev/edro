PASSO 1: INSTALAR NODE V20
---------------------------
1. Acesse: https://nodejs.org/
2. Baixe: Node.js v20.x LTS (Windows 64-bit)
3. Instale normalmente
4. Reinicie o PowerShell

Verificar:
  node --version
  (Deve mostrar: v20.x.x)


PASSO 2: CORRIGIR DEPENDENCIAS
-------------------------------
Execute:
  .\FIX_TUDO.ps1

Digite "s" quando perguntar.
Aguarde 5-10 minutos.


PASSO 3: CONFIGURAR .ENV
-------------------------
Edite o arquivo:
  apps\backend\.env

Configure:
  DATABASE_URL=postgresql://user:pass@host:5432/db
  PORT=3333
  JWT_SECRET=seu-secret-aqui
  OPENAI_API_KEY=sk-sua-key-aqui


PASSO 4: RODAR SERVIDOR
------------------------
Execute:
  cd apps\backend
  npm run dev

Acesse:
  http://localhost:3333


AJUDA RAPIDA
------------
Verificar instalacao:
  .\VERIFICAR_INSTALACAO.ps1

Ver proximos passos:
  notepad PROXIMOS_PASSOS.md

Ver erros encontrados:
  notepad PROBLEMAS_E_SOLUCOES.md


COMANDOS UTEIS
--------------
Limpar tudo:
  Remove-Item -Recurse -Force node_modules

Reinstalar:
  npm install

Rodar migrations:
  cd apps\backend
  npm run db:migrate
