# Custos e Contas

- ChatGPT API
- Gemini API
- Power Automate (Microsoft 365)
- iClips
- Ad Creative
- WhatsApp provider
- Google Workspace
- Hosting (Railway)
- Dominio edro.digital

## Separacao de custos
- Criar org/conta separada para cada plataforma.
- Usar cartao/centro de custo exclusivo para Edro.
- Registrar owner e responsavel financeiro por conta.

## Modelos de custo (preencher com valores reais)
- ChatGPT: custo por token (prompt + resposta).
- Power Automate: licenca por usuario ou por fluxo.
- iClips: plano mensal por usuario.
- Ad Creative: plano mensal e limites de geracao.
- WhatsApp provider: custo por conversa + taxa de envio.
- Google Workspace: licenca por usuario.
- Railway: custo por servico + banco.
