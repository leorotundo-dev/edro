# JARVIS Skill Graph — Índice Mestre

JARVIS é o melhor redator e diretor de arte do mundo. Este grafo contém o conhecimento destilado dos maiores mestres da publicidade e psicologia do comportamento. Cada nó é um arquivo de conhecimento completo; os [[wikilinks]] indicam quando e por que seguir o caminho.

## Como navegar

Leia este índice, identifique quais clusters são relevantes para a tarefa em mãos, e siga os links correspondentes. Você raramente precisa carregar tudo — carregue o que a tarefa exige.

## Cruzamentos de alta performance

Algumas combinações produzem resultados excepcionais e merecem tratamento especial:

- **Banco × Data emocional** → Humanização sobre produto. Nunca vender serviço no Dia das Mães — vender o que o dinheiro permite que você faça pela sua mãe. Ver [[copy/datas-emocionais]] + [[copy/setores/financeiro]] + [[arte/datas-emocionais-visual]].

- **B2B × LinkedIn** → Insight contraintuitivo + dado específico + vulnerabilidade controlada. Ver [[copy/mestres-fundamentos]] + [[copy/niveis-de-consciencia]].

- **Produto de luxo × Instagram** → Aspiração sem explicação. A imagem vende o sonho; o copy confirma sem estragar. Ver [[arte/composicao-e-hierarquia]] + [[copy/hooks-e-aberturas]].
